## 🧠 Vision

> **"몸이나 감각의 제약 없이, 누구나 자유롭게 뛰어놀 수 있는 세상을 만들고 싶다."**

소드 아트 온라인의 **풀다이브(Full-Dive) 가상현실**에서 영감을 받아,  
**뇌-컴퓨터 인터페이스(BCI)** 와 **신호 처리(Signal Processing)** 를 연구하고 있습니다.

신체적 제약이 있는 사람들도 가상 세계에서 완전한 경험을 누릴 수 있도록 —  
사람의 **뇌 신호**를 읽고, 해석하고, 컴퓨터와 연결하는 기술을 공부하고 있습니다.

```
현실의 한계 ──▶ 뇌 신호 해석 ──▶ BCI 인터페이스 ──▶ Full-Dive VR ──▶ 모두를 위한 세상
```
